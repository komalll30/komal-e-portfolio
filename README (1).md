# Komal Gowda – E-Portfolio

A personal portfolio website built with HTML and CSS, showcasing my academic background, technical skills, and projects as an Artificial Intelligence & Machine Learning student at REVA University.

🌐 **Live Site:** [https://komal-gowda-portfolio.vercel.app/](https://komal-gowda-portfolio.vercel.app/)

---

## About

This portfolio was created to present my journey as an AI & ML student, including my skills, education, and hands-on projects in data analytics and deep learning.

---

## Sections

- **Hero** – Introduction and quick navigation
- **About Me** – Background, interests, and focus areas
- **Skills** – Technical and professional skill bars
- **Education** – Academic timeline at REVA University
- **Projects** – Featured ML/AI projects with descriptions and tags
- **Contact** – Contact form and social links

---

## Projects Featured

| Project | Domain | Tech Stack |
|---|---|---|
| NeuroScope: MRI Tumor Detector | Computer Vision | Python, TensorFlow, Keras, OpenCV |
| CropWise: Plant Disease Classifier | Deep Learning | Python, PyTorch, FastAPI, Docker |
| InsightFlow: Analytics Dashboard | Data Analysis | Python, Pandas, Streamlit, NLTK |

---

## Tech Stack

- HTML5
- CSS3 (Glassmorphism design, CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Deployed on **Vercel**

---

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/e-portfolio.git
   ```
2. Open `index.html` in your browser — no dependencies or build step needed.

---

## Deployment

This site is deployed using [Vercel](https://vercel.com). Any push to the `main` branch automatically updates the live site.
